Instruções de Uso
🚀 Primeiros Passos
1. Configure o GitHub

Vá em ⚙️ Configurações, cole o token, usuário e repositório. Clique em "Salvar e Testar".

2. Troque qualquer imagem

Em 🖼️ Imagens, escolha a aba da página, clique em "Trocar Imagem" no card desejado e selecione a nova foto. Funciona para TODAS as imagens do site (incluindo logo, favicon e fundos).

3. Edite qualquer texto

Em ✏️ Textos, escolha a aba da página. Cada elemento de texto (título, parágrafo, link, botão…) aparece com um campo editável. Edite e clique em "Publicar".

4. Aguarde

O site atualiza em 1–2 minutos após cada publicação no GitHub Pages.

🖼️ Sobre as Imagens
O que é detectado

• Todas as tags <img> • Favicons (link rel=icon) • Imagens de fundo em style="background:url(...)"

Formatos aceitos

JPG, PNG e WEBP — máximo 5MB por imagem.

Substituição

A imagem é gravada no mesmo caminho original — todas as páginas que a usam são atualizadas automaticamente.

✏️ Sobre os Textos
O que é editável

Cada elemento da página que contém texto direto (h1, h2, h3, h4, p, span, a, li, button, strong, em, small, label).

Quebras de linha

Use a tag <br> dentro do texto para criar uma quebra de linha (igual ao HTML original).

HTML simples

O editor preserva tags inline como <br>, <strong>, <em>. Não cole HTML complexo.
